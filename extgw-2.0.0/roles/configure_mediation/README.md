#Ansible automation of External Gateway deployement on cloud servers - Role name : configure_mediation


##Important facts
* This role copy two files from local repository to the DAS server. They are inside the **files** folder of the role.
*  master-datasources.xml
*  mysql-connector-java-5.1.40-bin.jar
*  javax.persistence_1.0.0.jar
*  json-3.0.0.wso2v1.jar
*  mediator-conf.properties
*  MobileCountryConfig.xml
*  spendLimit.xml
